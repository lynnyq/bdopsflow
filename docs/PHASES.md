## 新增功能模块

### 1. DAG 工作流引擎
**位置**: `scheduler/internal/dag/dag.go`

**功能特性**:
- DAG 配置解析和验证
- 循环检测
- 拓扑排序（执行顺序）
- 节点依赖分析
- 自环和重复连接检测

**单元测试**: 11个测试用例，全部通过

### 2. Webhook 通知服务
**位置**: `scheduler/internal/webhook/webhook.go`

**功能特性**:
- 事件驱动的Webhook通知
- 支持自定义HTTP头
- 重试机制（指数退避）
- 事件过滤

**单元测试**: 6个测试用例，全部通过

### 3. 监控指标收集
**位置**: `scheduler/internal/metrics/metrics.go`

**功能特性**:
- Counter（计数器）
- Gauge（仪表）
- Histogram（直方图）
- Redis持久化
- 后台定时收集

**单元测试**: 13个测试用例，全部通过

## 快速开始

### 后端启动
```bash
cd scheduler
go build -o bin/scheduler ./cmd
./bin/scheduler
```

### 前端启动
```bash
cd web
npm install
npm run dev
```

### 运行测试
```bash
cd scheduler
go test ./internal/... -v
```

### Docker部署
```bash
docker-compose up -d
```
